# 📊 Streamlit RAG Data Analyst

A Retrieval-Augmented Generation (RAG) Business Analytics Assistant built with:

- Streamlit
- Pandas
- Plotly
- GPT-4.1 Mini
- GitHub Models API

---

## 🚀 Live Preview

👉 **[Launch App (Streamlit Preview)](https://app-rag-data-analyst-pk9ohfsmd8ps2hfxwpqwbp.streamlit.app/)**

---

## ✨ Features

- 📂 Upload CSV files
- 📈 Interactive sales charts
- 🔎 RAG-based data retrieval
- 🧠 AI-powered business insights
- ⚠️ Risk detection
- 💡 Opportunity analysis
- 💬 Chat with your dataset
- 🛡️ Prompt injection protection layer

---

## 🧠 How it works

1. User uploads sales CSV
2. System retrieves relevant data (RAG layer)
3. LLM (GPT-4.1 Mini / LLaMA) analyzes context
4. AI returns:
   - Insights
   - Risks
   - Opportunities
   - Recommended charts
