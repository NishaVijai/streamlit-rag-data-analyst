# Streamlit RAG Data Analyst

A Retrieval-Augmented Generation (RAG) Business Analytics Assistant built with:

- Streamlit
- Pandas
- Plotly
- GPT-4.1 Mini
- GitHub Models API

## Features

- Upload CSV files
- Interactive sales charts
- RAG-based retrieval
- Business insights
- Risks and opportunities analysis
- Chat with your data

## Run Locally

pip install -r requirements.txt

streamlit run streamlit_rag_analysis_prompt_app.py
